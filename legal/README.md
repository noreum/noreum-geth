# Noreum Legal Documentation

This directory contains all legal and licensing materials for the **Noreum Execution Client (noreum-geth)**, a modified and re-engineered version of the Ethereum client (Geth).

---

##  Overview

The Noreum Execution Client is a derivative work based on **go-ethereum (Geth)**, originally licensed under the **LGPL-3.0** license.  
All modifications, enhancements, and new components created by the **Noreum Core Team** are licensed under **GPL-3.0**.

This dual-layer structure ensures that:
- All original Ethereum (Geth) authors retain credit.
- All new Noreum code remains open-source and publicly auditable.
- Redistribution complies with both LGPL and GPL requirements.
